# 3MTT Abia Pulse (Offline-First PWA)
**Developer:** Chimgozirim
**Track:** Front-End Development / Digital Inclusion

## Overview
A minimalist, Brutalist-inspired directory for 3MTT fellows and community members. 
This app is designed to solve "Data Poverty" by allowing users to access 
vital information without an active internet connection.

## Key Features
- **Offline-First:** Uses Service Worker API to cache resources.
- **Brutalist UI:** High-contrast, zero-clutter design for readability and speed.
- **PWA Ready:** Installable on Android and iOS devices.

## Tech Stack
- HTML5 / CSS3
- Vanilla JavaScript
- Service Worker API

## How to use
1. Visit the live link.
2. Allow the page to load once.
3. Turn off your data/Wi-Fi.
4. Refresh or navigate the site to see the offline capability in action.
